# FinVision - Finance Dashboard

A clean, interactive finance dashboard built with vanilla HTML, CSS, and JavaScript. Features real-time data visualization, transaction management, and role-based access control.



## ✨ Features

### Core Requirements ✅
- **Dashboard Overview**: Summary cards for Total Balance, Income, and Expenses
- **Time-based Visualization**: Balance trend chart (last 6 months)
- **Categorical Visualization**: Spending breakdown by category (doughnut chart)
- **Transaction Management**: Complete CRUD operations with filtering, search, and sorting
- **Role-based UI**: Admin (full control) vs Viewer (read-only) with toggle switch
- **Smart Insights**: Highest spending category, monthly comparison, and data observations

### Additional Enhancements 🎯
- 🌙 **Dark Mode** - Toggle between light and dark themes
- 💾 **Local Storage** - Data persistence across browser sessions
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile devices
- 🎨 **Modern UI** - Clean, intuitive interface with smooth animations
- 🔍 **Advanced Filtering** - Search by description/category, filter by type and category
- 📊 **Dynamic Charts** - Real-time chart updates when data changes

## 🛠️ Technologies Used

- **HTML5** - Semantic structure
- **CSS3** - Flexbox, Grid, custom properties, responsive design
- **JavaScript (ES6+)** - Vanilla JS, no frameworks
- **Chart.js** - Data visualization library
- **Font Awesome 6** - Icons
- **Google Fonts** - Inter font family
